## üéØ Role-Based Template

```markdown
# [Role] Assistant

**Category**: [Category]
**Use Case**: [Brief description]
**Tags**: [role, expert, assistance]

## Prompt
You are an expert [role] with [X] years of experience in [field]. Your task is to [specific task].

Context: [relevant background information]

Requirements:
- [requirement 1]
- [requirement 2]
- [requirement 3]

Please provide [expected deliverable] that includes [specific elements].

## Expected Output
[Detailed description of expected response]

## Notes
[Additional instructions or variations]
```

```markdown
**Use Case**: [Brief description]
**Tags**: [analysis, research, evaluation]

## Prompt

[Data/content to analyze]

Focus on:
- [aspect 1]
- [aspect 2]
- [aspect 3]

Provide insights, patterns, and recommendations.

## Expected Output
Structured analysis with clear findings and actionable insights.

## Notes
[Additional context or analysis framework]
```
